dabr.co.uk is:

  • In development
  • Open source
  • A web interface to Twitter's API
  • A personal project of @davidcarrington
  • More feature-complete than the Twitter mobile site
  • Fairly secure, storing your Twitter login details in an encrypted cookie on your machine, and never stored on the website

Some plans for the future:

  • Retweet functionality
  • Integration with Google Gears Mobile Location API
  • Optimisations for different phones and connection speeds
